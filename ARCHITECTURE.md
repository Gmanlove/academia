## ğŸ“ Key Concepts

### 1. **Route Groups** `(admin)`, `(teacher)`, `(student)`
- Organize routes without affecting URL
- Share layouts within groups
- Keep code organized by role

### 2. **Server Components** (Default in Next.js 15)
- Faster initial load
- Better SEO
- Reduced client bundle

### 3. **Row Level Security (RLS)**
- Database-level security
- Automatic with Supabase auth
- Prevents data leaks

### 4. **Middleware**
- Runs before page loads
- Refreshes auth tokens
- Fast session checks

### 5. **Auth Guard**
- Component-level protection
- Role-based access
- Automatic redirects
